# Philosophy Vault

Obsidian-alapú filozófiai jegyzetgyűjtemény.

## Tartalom

- **CC-Philosophy**: A Crash Course Philosophy sorozat magyar nyelvű feldolgozása (~45 epizód)
- **Buddhista filozófia**: Pratītyasamutpāda és kapcsolódó témák

## Struktúra

A `Kidolgozott` mappában találhatók a véglegesített, kidolgozott jegyzetanyagok. Minden fájl egy-egy téma önálló feldolgozása, Obsidian markdown formátumban.
