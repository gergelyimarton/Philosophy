# 21 – Személyiség (Personhood)

> **Eredeti cím:** Personhood: Crash Course Philosophy #21
> **Kulcsszavak:** személyiség, morális közösség, genetikai kritérium, kognitív kritériák, gradiens elmélet

---

## 1. Ember vs. Személy

### 1.1 A megkülönböztetés

| Fogalom | Típus | Meghatározás |
|---------|-------|--------------|
| **Ember (Human)** | Biológiai | Emberi DNS-sel rendelkezik |
| **Személy (Person)** | Morális | Morális közösségünk tagja |

### 1.2 Miért fontos?

```
Lehet valaki:
- Ember, de nem személy? (pl. magzat?)
- Nem ember, de személy? (pl. Superman?)
```

## 2. Superman-teszt

### 2.1 A kérdés

> Superman személy?

### 2.2 Elemzés

| Tény | Következtetés |
|------|---------------|
| Nem emberi DNS (kriptoni) | Biológiailag NEM ember |
| Beszél, gondolkodik, érez | Viselkedése személyé |
| Lex Luthor gyűlöli, mert "idegen" | → Lex előítéletes |
| Lois, Jimmy, a Kenték személynek tekintik | → Számukra személy |

→ **Ha Superman személy, akkor a DNS nem a lényeg**

## 3. Kritériumok a személyiséghez

### 3.1 Genetikai kritérium (John Noonan)

> **Személy = emberi DNS**

| Előny | Hátrány |
|-------|---------|
| Egyszerű | Szájsejtek is "személyek"? |
| Egyértelmű | Holttestek is? |
| | Superman, WALL-E kizárva |

### 3.2 Kognitív kritériumok (Mary Ann Warren)

5 feltétel **együtt**:

| # | Kritérium |
|---|-----------|
| 1 | Tudat (consciousness) |
| 2 | Érvelés (reasoning) |
| 3 | Önmotivált aktivitás |
| 4 | Kommunikáció képessége |
| 5 | Öntudat (self-awareness) |

**Probléma:**
- Magzatokat kizárja ✓ (Warren szerint)
- Kisgyerekeket is kizárja ✗ (öntudat ~18 hónaptól)

### 3.3 Szociális kritérium

> **Személy vagy, ha a társadalom annak ismer el / valaki törődik veled**

| Előny | Hátrány |
|-------|---------|
| Intuitív | Ha senki sem törődik veled? |
| Változhat az idővel | Népszerűségi verseny? |
| Pl. főemlősök bevonhatók | Magányos emberek nem személyek? |

### 3.4 Érzőképesség kritérium (Peter Singer)

> **Személy = ami képes fájdalmat és örömöt érezni**

```
Fájdalmat okozni = rossz
Ha nem tud érezni = nem okozunk kárt
```

| Személy | Nem személy |
|---------|-------------|
| Fejlett idegrendszerű állatok | 23 hét alatti magzat |
| Felnőtt emberek | Vegetatív állapotú ember |

### 3.5 Személyiség elvesztése

Bizonyos nézetek szerint a személyiség **elveszíthető**:
- Súlyos bűncselekmény (gyilkosság, nemi erőszak)
- → A halálbüntetés igazolása

## 4. Gradiens elmélet

### 4.1 A modell

> A személyiség **nem kapcsoló, hanem dimmer** – fokozatai vannak

```
Személyiség foka:

magzat → újszülött → csecsemő → kisgyerek → felnőtt
   ↓         ↓          ↓          ↓          ↓
  alacsony  ...        ...        ...       magas
```

### 4.2 Alkalmazás

| Helyzet | Következtetés |
|---------|---------------|
| Magzat vs. anya | Anyának több személyisége → érdekei fontosabbak |
| Demens idős | Csökkenő személyiség |
| Nem tagadja egyik fél személyiségét | De súlyozza |
