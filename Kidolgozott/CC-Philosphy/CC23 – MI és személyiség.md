## 1. Gyenge vs. Erős MI

### 1.1 Gyenge MI (Weak AI)

> **Emberi intelligencia egyes aspektusait utánozza**

| Példák |
|--------|
| Siri, Alexa |
| Helyesírás-ellenőrzés |
| Számológép |
| Útvonaltervezés |

→ Korlátozott képességek

### 1.2 Erős MI (Strong AI)

> **Valóban gondolkodik, mint mi**

```
Nem csak utánoz → ténylegesen gondolkodik
Nem csak szimulál → valódi megértés
```

→ Még nem létezik

## 2. Turing-teszt (Alan Turing, 1950)

### 2.1 A teszt

```
1. Szöveges beszélgetés két "személlyel"
2. Egyik ember, másik gép
3. Nem tudod, melyik melyik
4. Bármit kérdezhetsz

KÉRDÉS: Meg tudod különböztetni őket?
```

### 2.2 Turing állítása

> Ha a gép meg tud téveszteni → **erős MI-vel rendelkezik**

### 2.3 Az érv

```
Ha nem tudod megkülönböztetni → nincs különbség
A viselkedés a mérce
Ahogy egymást is viselkedés alapján ítéljük meg
```

## 4. Kínai szoba (John Searle)

### 4.1 A gondolatkísérlet

```
1. Nem beszélsz kínaiul
2. Bezárnak egy szobába
3. Kapsz:
   - Dobozok kínai karakterekkel
   - Angol nyelvű kódkönyv (melyik karakterre mit válaszolj)
4. Kívülről kínai üzeneteket küldenek be
5. A könyv alapján válaszolsz (nem értesz semmit)
6. A kínaiak azt hiszik, tudsz kínaiul

KÉRDÉS: Tudsz kínaiul?
```

### 4.2 Searle válasza

> **NEM** – csak szimbólumokat manipulálsz, nincs megértés

### 4.3 Következtetés

```
Turing-teszt átmenése ≠ erős MI
A megtévesztés képessége ≠ valódi gondolkodás
Szükséges: TÉNYLEGES MEGÉRTÉS
```

---

## 5. Ellenérv: A rendszer érvel

### 5.1 Az érv

```
Te egyedül nem értesz kínaiul
DE: Az egész rendszer (te + könyv + szimbólumok) TUDJA
```

### 5.2 Analógia

| Az agyad egyes részei | Az egész agyad |
|-----------------------|----------------|
| Nem beszélnek angolul | Beszél angolul |

→ A **rendszer egésze** rendelkezhet megértéssel
