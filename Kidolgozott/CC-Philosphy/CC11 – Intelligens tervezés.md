## 1. A teleológiai érv

### 1.1 Alapfogalom

> **Teleológia** (görög: telos = cél) = célra irányultság, tervszerűség

### 1.2 Az érv lényege

Ha a világban **célszerűséget** látunk → kell lennie **tervezőnek**

---

## 2. William Paley (1743–1805)

### 2.1 Az órás-analógia

```
HELYZET: Találsz egy órát a mezőn

KÉRDÉS: Véletlenül jött létre, vagy valaki készítette?

VÁLASZ: Nyilvánvalóan készítették
        → A komplexitás és célszerűség TERVEZŐT feltételez
```

### 2.2 Az érv kiterjesztése

| Óra | Emberi test / Természet |
|-----|------------------------|
| Fogaskerekek illeszkednek | Szervek együttműködnek |
| Célja van (időmérés) | Célja van (életben maradás) |
| Készítette valaki | Készítette valaki (Isten) |

### 2.3 Formális rekonstrukció

```
P1: Az óra komplex és célorientált → készítője van
P2: A természet (emberi test, szem, stb.) is komplex és célorientált
────────────────────────────────────────────────────────────────────
K:  A természetnek is van készítője → Isten
```

## 3. Ellenvetések

### 3.1 Disanalógia

> Az analógia akkor működik, ha A és B elég hasonlóak.

| Óra | Természet |
|-----|-----------|
| Ismerjük a készítőjét | Nem ismerjük |
| Egyszerűen érthető | Sok rejtély |
| Nincs "hibás" része | Vakfolt, vestigális szervek |

### 3.2 Céltalan részek

| "Hiba" | Leírás |
|--------|--------|
| Vakfolt | Nincs funkciója |
| Férfi mellbimbó | Nincs funkciója |
| Vakbél | Csökevényes szerv |

**Kérdés:** Ha Isten tervezte, miért vannak hibák?

### 3.3 Mi találunk célt, vagy ott van?

**Probléma:** Bármiben "találhatunk" célt

| Testrész   | "Cél"                |
| ---------- | -------------------- |
| Ujj        | Orrtúrás?            |
| Nyúl farka | Célpont vadászoknak? |

→ A célt **mi vetítjük ki**, nem benne van

### 3.4 Evolúció – alternatív magyarázat

**Darwin (1859):** A komplexitás magyarázható **természetes szelekcióval**

| Tervező | Evolúció |
|---------|----------|
| Szándékos létrehozás | Véletlen mutáció + szelekció |
| Egyszeri teremtés | Fokozatos fejlődés |
| Cél-vezérelt | Cél nélküli folyamat |

### 3.5 Hibás tervező (Hume)

David Hume kritikája:

> Ha komolyan vesszük az analógiát, a tervező **hibásnak** tűnik.

| "Hibák" a természetben |
|------------------------|
| Hurrikánok, földrengések |
| Rák, betegségek |
| Ragadozók, szenvedés |

→ Ha van tervező, nem tökéletes

## 4. Modern változatok

### 4.1 Fine-tuning érv

```
P1: Az univerzum "finomhangolt" az életre
P2: Ha a fizikai állandók kicsit mások → nincs élet
P3: Ez nem valószínű, hogy véletlen
────────────────────────────────────────────
K:  Valaki finomhangolta → Isten
```

### 4.2 Ellenvetés

- Csak **egy** univerzumunk van → nem számolhatunk valószínűséget
- **Antropikus elv:** Csak olyan univerzumban kérdezhetjük ezt, ahol van élet
